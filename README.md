# ElectroPro Solutions

Web profesional de electricidad para viviendas, industria, renovables y naval.

## Características
- Diseño minimalista corporativo (negro, rojo, blanco)
- 100% responsiva
- Multilenguaje (ES/EN)
- Formulario funcional (Formspree)
- WhatsApp flotante
- Galería de proyectos
- SEO avanzado (meta, sitemap, schema)
- Google Analytics
- SSL automático (GitHub Pages)

## Instalación
1. Sube todos los archivos a tu repositorio
2. Activa GitHub Pages en `main` → `/ (root)`
3. ¡Listo!

## Personalización
- Cambia `YOURUSERNAME` en todos los archivos
- Configura Formspree: [https://formspree.io](https://formspree.io)
- Añade tu ID de Google Analytics
